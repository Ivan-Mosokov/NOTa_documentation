## Overview

Сервис мониторинга (Monitoring) обеспечивает комплексный контроль производственных процессов в реальном времени. Система включает мониторинг ресурсов, оборудования, планирование ТОиР, проектирование 2D модели производства и аналитику для принятия управленческих решений.

## Основные функции

### Мониторинг ресурсов системы в реальном времени
- Отслеживание загрузки буферов
- Мониторинг производственных участков
- Контроль загрузки операторов
- Отслеживание логистических маршрутов

### Мониторинг оборудования и управление ТОиР
- Отслеживание состояния оборудования в реальном времени
- Мониторинг температуры, вибрации, загрузки
- Планирование профилактического обслуживания
- Интеграция с системами IIoT для прогнозного анализа отказов

### Проектирование 2D модели производства
- Стандартизация расположения оборудования
- Проектирование логистических маршрутов
- Внедрение принципов OCP (НОТа)
- Сокращение времени внедрения и повышение производительности

### Аналитика и визуализация данных
- Формирование отчетов по ключевым показателям (OEE, время простоя, себестоимость)
- Настраиваемые параметры отчетности
- Инструменты для анализа причинно-следственных связей
- Влияние качества сырья на брак и другие метрики

## Архитектура

Сервис построен на микросервисной архитектуре и взаимодействует с другими компонентами системы через события и API.
