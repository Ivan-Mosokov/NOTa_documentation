## Overview

Task Manager представляет собой канбан доску для управления задачами операторов. Для каждого сотрудника отображаются задачи, которые он может управлять, просматривать технологическую информацию, отправлять файлы на ЧПУ станки, фиксировать начало выполнения задачи, окончание, простой, перерыв. Система оснащена автоматическими напоминаниями при превышении времени выполнения операций или перерывов.

## Основные функции

### Канбан доска
- Визуализация задач для каждого оператора
- Drag & drop управление задачами
- Отображение статуса выполнения
- Приоритизация задач

### Управление задачами
- Просмотр технологической информации
- Отправка файлов на ЧПУ станки
- Фиксация начала выполнения задачи
- Фиксация окончания выполнения
- Учет простоев и перерывов

### Система напоминаний
- Автоматические напоминания о перерывах
- Уведомления о превышении времени выполнения
- Контроль соблюдения трудового законодательства
- Настройка параметров уведомлений

### Интеграция с оборудованием
- Отправка программ на ЧПУ станки
- Получение статуса выполнения операций
- Синхронизация с системой мониторинга
- Контроль состояния оборудования

## Архитектура

Сервис построен на микросервисной архитектуре и взаимодействует с другими компонентами системы через события и API.
