Шаг логистической операции — минимальное действие в процессе перемещения. Бывает двух типов:

- **Акт** — выполняется один раз для всей партии (например, погрузка контейнера).
- **Цикл** — повторяется для каждого изделия или группы изделий (например, перевозка деталей поштучно).
