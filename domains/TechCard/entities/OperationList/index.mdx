## Overview
OperationList представляет технологические операции, которые необходимо выполнить для производства изделия согласно техкарте. Каждая операция содержит детальную информацию о процессе, оборудовании, времени выполнения и требованиях.

### Взаимосвязи:
- **TechCard → OperationList**: Один ко многим - техкарта содержит множество технологических операций
- **OperationList → Routing**: Один ко многим - технологическая операция может содержать множество логистических операций
- **OperationList → Equipment**: Многие к одному - операция выполняется на конкретном оборудовании
- **OperationList → Workplace**: Многие к одному - операция выполняется на конкретном рабочем месте
