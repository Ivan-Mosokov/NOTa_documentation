## Overview

Сервис реестра техкарт (Tech Card Register) обеспечивает централизованное управление технологическими картами, предоставляя функциональность для регистрации, поиска и контроля версий техкарт. Этот сервис является ключевым компонентом системы технологической подготовки производства.

<NodeGraph />

## Основные функции

### Реестр техкарт
- Регистрация новых техкарт в системе
- Поиск и фильтрация техкарт по различным критериям
- Управление версиями техкарт
- Контроль статуса техкарт (черновик, опубликована, архив)

### Управление жизненным циклом
- Отслеживание изменений в техкартах
- Контроль процесса утверждения и публикации
- Архивирование устаревших версий
- Аудит изменений и действий пользователей

### Интеграция
- Связь с редактором техкарт
- Интеграция с системой планирования
- Экспорт данных для других систем
- Синхронизация с внешними реестрами

## Архитектура

Сервис построен на микросервисной архитектуре и взаимодействует с другими компонентами системы через события и API.
