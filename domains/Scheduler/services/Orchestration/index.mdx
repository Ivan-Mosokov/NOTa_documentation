## Overview

Сервис оркестрации (Orchestration) обеспечивает построение взаимосвязанного расписания задач для операторов. Интерфейс представляет собой диаграмму Ганта с возможностью drag & drop перетаскивания задач, автоматическим сдвигом связанных операций и визуализацией конфликтов.

<NodeGraph />

## Основные функции

### Диаграмма Ганта
- Визуализация расписания в виде диаграммы Ганта
- Список операторов в левой части интерфейса
- Лента времени для планирования задач
- Drag & drop перетаскивание задач из ProcessPlan

### Управление зависимостями
- Автоматический сдвиг связанных последующих задач
- Учет технологических зависимостей между операциями
- Визуализация критического пути
- Оптимизация последовательности выполнения

### Конфликт-менеджмент
- Визуализация пересечений задач (красная обводка)
- Автоматическое обнаружение конфликтов расписания
- Предложения по разрешению конфликтов
- Валидация корректности расписания

### Интеграция с Task
- Отправка времени начала и окончания операций
- Синхронизация с системой управления задачами
- Обновление статусов выполнения
- Контроль соблюдения расписания

## Архитектура

Сервис построен на микросервисной архитектуре и взаимодействует с другими компонентами системы через события и API.
