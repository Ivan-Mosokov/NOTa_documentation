Процессный план (ProcessPlan) — список всех операций, необходимых для выполнения производственного плана.

- Включает технологические и логистические операции, которые подтягиваются из техкарты.
- Описывает порядок их выполнения в рамках заказа.
- Используется для планирования, распределения и контроля операций.
